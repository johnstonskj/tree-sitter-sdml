1.4. Entities

Figure 7: Entity Stereotype

Figure 8: Entity Member Stereotypes

For details on by-value members and groups see section 1.7 below.

1.4.1. Example Usage

module rentals is
  import uuid

  datatype VehicleIdNumber <- string

  entity Vehicle is
    identity vin -> VehicleIdNumber
  end

  entity Booking is
    identity id -> uuid::Uuid
    ref vehicle {0..1} -> {0..1} Vehicle
  end

end
